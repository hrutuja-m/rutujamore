ShootVision: AI-powered Basketball Shot Analysis

The goal of the project was to develop a machine learning-based solution for analyzing basketball shots and shooting pose. The solution involved implementing an object detection algorithm, to extract more detailed information about the shots and shooting pose, the OpenPose API was integrated with the machine learning model. This allowed for the analysis of subtle movements and positions of the body, which are crucial for identifying shooting technique and form.
